A Chrome extension that acts as an AI mentor on LeetCode, guiding you with Socratic questions instead of giving answers.

## Setup

1. **Get a Groq API key** at [console.groq.com/keys](https://console.groq.com/keys) (free tier available).

2. **Create your config file:**
   - Copy `config.example.js` to `config.js`
   - Open `config.js` and replace `PASTE_YOUR_GROQ_API_KEY_HERE` with your Groq API key

3. **Load the extension in Chrome:**
   - Go to `chrome://extensions`, enable Developer mode
   - Click "Load unpacked" and select the `extension1` folder

4. **Use it:** Open any LeetCode problem page and click the extension icon.
